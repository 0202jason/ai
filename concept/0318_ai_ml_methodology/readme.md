## 선형 회귀(simple linear regression)

**선형회귀란?**
- 입력 변수, 출력 변수 사이의 관계를 직선 형태로 근사, 예측하는 통계적 방법

## 단순 선형 회귀

**단순선형회귀(simple linear regression)란?**
- 한 개의 설명변수(X)와 하나의 반응변수(Y) 사이의 선형(직선) 관계를 찾는 방법
- 데이터를 가장 잘 설명하는 직선을 찾아 예측(y^)에 활용
> Y = β0 + β1X + ε
    - β0: 절편, β1: 기울기, ε: 관측 오차
    - ^ 표기는 추정값을 의미

### 최소제곱법(least squares)
- 실제 관측값과 예측값의 차이(잔차, residual)를 제곱해 합한 값(RSS, 잔차제곱합)을 최소화하는 방법
