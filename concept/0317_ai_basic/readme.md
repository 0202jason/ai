## AI, ML, DL 정의

**AI**: 주어진 환경, 데이터를 인지 학습 추론을 통해 목표 달성을 하도록 예측, 행동 선택, 계획 하는 시스템
**ML(machine learning)**: AI 범주 내에서 데이터로부터 학습하여 목적을 달성하는 접근 방법론
  - 예: 생성형 ai, 언어모델, 이미지 분류 모델, 추천 시스템
**DL(deep learning)**: ML 범주 내에서 신경망(Neural Network)함수를 사용한 학습 방법론


## 데이터 구성 요소

- 데이터가 중요한 이유: 머신러닝은 규칙을 직접 코딩 x 데이터에서 규칙을 학습. 데이터의 분포, 관계가 머신러닝의 학습 결과를 결정
- Feature(피처, 특성): 모델이 예측에 사용하는 입력정보. 예측, 판단의 근거
- Label(라벨, 목표값): 모델이 예측하려는 정답. 학습의 목표값

## ML 예시 

- 유튜브 추천
  - Feature: 영상의 정보(장르, 조회수, 좋아요, 크리에이터), 사용자 정보(시청 이력, 구독 채널)
  - Label: 영상에 대한 사용자 피드백(시청 여부, 좋아요)
- 스팸메일 분류
  - Feature: 메일 제목, 발신자, 단어 빈도
  - Label: 스팸/ 정상

## 1D 피쳐 기반 학습

- 1D = 1차원
- Feature가 하나일 때 머신러닝이 학습하는 가장 단순한 형태
> Income = f(Years of Education) + ε

#### 미지의 함수(f): feature와 label 사이의 실제 평균 관계
- 직접 관측 안됨
- 오차가 포함된 데이터(점)만 관측 가능
  
#### 측정 오차(ε): 데이터에 섞인 오차
- 원인: 측정 기기의 한계, 환경적 요인
- 따라서, 데이터 = 참 함수 + 오차


## 모델과 가설 공간
#### 학습
- 입력(Feature) > 출력(Label) 관계를 찾는 과정
- 평균 관계를 하나의 함수로 표현
- 관계를 표한할 수 있는 함수는 무수히 많음

#### 가설공간(Hypothesis Space)
- 관계를 표현할 수 있는 모든 후보 함수들의 모음
- 피쳐 공간과 라벨 공간위에서 정의된 함수의 집합

#### 모델(Model)
- 가설공간에 속한 특정 함수 f

## 학습이란
#### 학습
- 주어진 데이터에서 정답을 가장 잘 맞출 수 있도록 모델의 규칙을 조금씩 조정해가는 과정
- 데이터 D -> 가설공간 F -> 선택된 모델 f
  
#### 학습에 필요한 3가지
1. 데이터(Data): 학습할 예시들
2. 가설공간(Hypothesis Space): 선택할 수 있는 모든 후보 함수들의 집합
3. 선택 기준(손실 함수): 어떤 함수가 더 좋은지 판단하는 척도. 예측값과 실제값의 차이 측정

#### 학습 과정
1. 가설공간에서 하나의 함수를 선택
2. 그 함수로 데이터의 모든 예시를 예측
3. 손실함수로 틀린 정도 계산
4. 더 적게 틀리도록 함수의 파라미터 조정
5. 반복하여 최적의 모델 완성

## 2D 피쳐 기반 학습
> Income = f(Years of Education, Seniority) + ε

E(ε)=0

## 왜 f()를 학습하는가?
#### 예측
- 잘 학습된 f가 있으면 새로운 입력 X = x에서 반응/목표 Y를 예측할 수 있음

#### 중요 특성 파악
- 피쳐들 X = (X1, X2, ... ,)중 어떤 특성이 Y를 설명하는데 중요하고 덜 중요한지 알 수 있음

#### 해석 가능성
- f의 복잡도에 따라 각 구성요소 Xi가 Y에 어떻게 영향을 미치는지 이해할 수 있음
